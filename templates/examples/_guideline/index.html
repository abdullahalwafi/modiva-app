Welcome to UI/UX Guideline

Ini adalah halaman dokumentasi yang dapat dilihat ketika terdapat "setting.INCLUDE_EXAMPLES == True".

Dokumentasi ini terdapat pada "templates/examples" dimana nanti akan terdapat folder "_guideline" untuk isi content dokumentasi ini dan "pages" untuk cara penggunaan komponennya. Adapun untuk views nya terdapat pada "module/examples" dimana akan sebagai contoh juga bagaimana memasukan data pada komponen.

Untuk komponen, terdapat pada folder "components" dimana nantinya bisa dipanggil melalui { % components table data=table % } (referensi dari django-components)

Untuk selanjutnya, anda dapat memulai dengan navigasi di sebelah kiri.
