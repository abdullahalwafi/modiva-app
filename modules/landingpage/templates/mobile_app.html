Aplikasi Mobile MODIVA untuk Siswi

Mobile App MODIVA dirancang untuk membantu siswi mengakses informasi terkait distribusi vitamin penambah darah secara lebih mudah, praktis, dan terarah melalui perangkat mobile.

Melalui aplikasi ini, siswi dapat memperoleh informasi pendukung mengenai program distribusi, edukasi kesehatan, serta pemantauan yang berkaitan dengan konsumsi vitamin penambah darah.

Halo, Siswi! 👋

Ayo jaga kesehatan tubuh dengan rutin mengonsumsi Tablet Tambah Darah (TTD).

Target Konsumsi Aktif
3 dari 4 Tablet 75%
Minum TTD setelah makan dengan air putih atau jus buah agar penyerapan maksimal.

Informasi Program

Menampilkan informasi terkait program distribusi vitamin penambah darah bagi siswi secara real-time dan terarah.

Akses Praktis

Memudahkan siswi mengakses informasi, mencatat riwayat konsumsi, dan mendapat notifikasi melalui perangkat mobile secara cepat.

Edukasi Kesehatan

Mendukung pemahaman siswi mengenai pentingnya konsumsi vitamin penambah darah demi mencegah anemia dan menjaga kebugaran tubuh.
